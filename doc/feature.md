# UniBridge – Feature Overview

## 1. University Finder
Search universities worldwide using smart filters such as location, programs offered, and acceptance rate. Each result includes a shortcut button that takes you directly to the AI Counselor with a pre-filled prompt like “Tell me more about [University]”.

## 2. AI Counselor
A conversational assistant that provides personalized guidance on admissions, programs, and applications. Features smooth line-by-line scrolling for a natural chat experience.

## 3. SOP Rater
Evaluates Statements of Purpose ethically, highlighting clarity, storytelling, structure, and alignment with your academic goals. Does NOT generate essays, only provides constructive feedback.

## 4. Application Reviewer
Reviews student applications, including resumes, activities, and achievements. Highlights strengths, identifies gaps, and suggests improvements for better alignment with university expectations.

## 5. Auto-Scroll & Smooth Chat UX
Ensures chat responses appear line by line, automatically scrolling for a seamless, human-like conversation.
